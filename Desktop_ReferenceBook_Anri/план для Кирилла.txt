================================ Редактирование ================================
страница добавления новой / редактирования компании (редактирвоание):
	- поле названия компании
	- выбрать из списка категорий (должны быть в базе)
	- выбрать из списка под-категорий (должны быть в базе)
	- текстовое поле для короткого описания
	- текстовое поле для полного описания
	- поле ФИО директора
	- список добавляемых телефонов
	- список email
	- список лого/фотографий (загрузка новых фото) 
		+ выбор из них фото как фото профиля компании (логотип)
	- поле для ссылки на сайт
	- список социальных цетей с полями ввода на профили в соц-сетях
	- адрес:
		- выбрать из списка город
		- выбрать из списка индекс
		- выбрать улицу
		- выбрать номер дома или поле номера (номер дома)
		- выбрать или поле номера копуса (пустое если нет)
		- поле квартира/офис
		- поле расписание

================================ Отображение ================================
страница отображения одной компании
	- вывести содержимое о компании

страница со выводом списка компаниий (титульная)
страница со выводом списка компаниий (фильтр поиска)
страница со выводом списка городов
	+ ajax добавление нового / редактирование города
страница со выводом списка индексов города
	+ ajax добавление нового / редактирование индекса города
страница со выводом списка улиц города
	+ ajax добавление нового / редактирование улицы города

по поводу ajax спросить Лешу. Нужно для того, чтобы не создавать пустые лишние страницы
